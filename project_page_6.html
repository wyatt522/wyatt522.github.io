Project Summary

I am currently creating a program to simultaneously map a robots enviroment, localize itself within that enviroment, and navigate throughout it. The robot currently navigates through an enviroment using the A* pathfinding algorithm. Using ROS, the program can be run on either the physical robot, or a corresponding gazebo simulation.

Turtlebot

I am using a Turtlebot 3 robot and packages to implement SLAM. We will use the turtlebot's LIDAR sensor in order to collect data for localization and mapping.
